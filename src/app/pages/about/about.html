About Us

Welcome to this demo webshop! This project was created for portfolio purposes to demonstrate a full-stack e-commerce application using Spring Boot, MySQL, JWT authentication, and Angular.

Purpose

The webshop is for demonstration only. No real transactions are processed, and no personal data is stored. It showcases frontend and backend integration, product listings, search functionality, authentication, and a full UI/UX layout.

Technologies Used

  • Spring Boot / Spring Data JPA
  • MySQL Database
  • JWT Authentication
  • Angular with TypeScript
  • Bootstrap for styling
